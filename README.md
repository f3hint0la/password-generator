# Random Password Generator

An online tool built with React JS to generate random password.

# Features

- Uppercase
- Lowercase
- Numeric
- Special Characters

### Installation

#### Clone repo

```bash
git clone https://github.com/f3hint0la/password-generator.git
```

#### Install dependencies

```bash
npm install
```

or,

```bash
yarn
```

#### Run the application

```bash
npm run start
```

or,

```bash
yarn run start
```
